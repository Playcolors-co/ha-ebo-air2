# Playcolors.co — Beta / test channel

⚠️ **Unstable — for testing only.** This repository is the **beta/test channel** for
Playcolors.co Home Assistant add-ons: a shared place to try changes **before** they are
released. Anything here can change, break, or be removed at any time.

**Stable releases live in their own repositories** (one per project). Add those for normal use:

| project | stable repository |
|---------|-------------------|
| **Enabot (EBO robots)** | https://github.com/Playcolors-co/ha-enabot |
| _…future projects…_ | _their own repos_ |

## Add-ons currently in testing here

| add-on | what it is | stable repo |
|--------|-----------|-------------|
| **EBO Air 2 (beta)** | Enabot EBO Air 2 — control, telemetry & camera | [ha-enabot](https://github.com/Playcolors-co/ha-enabot) |

## How to use

Only add this repository if you want to test **pre-release** builds:

```
https://github.com/Playcolors-co/ha-addons-beta
```

For normal use, add the **stable** repository of the add-on you want instead (see the table
above). Beta and stable add-ons can share the same entities, so run **one at a time**.
