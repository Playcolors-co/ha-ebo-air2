# Playcolors.co — Home Assistant add-ons

A collection of **Home Assistant add-ons** by Playcolors.co. Add this repository once and any
add-on it contains (now and in the future) shows up in your add-on store.

> ⚠️ Independent, unofficial projects. Not affiliated with the vendors of the devices they
> integrate. They interoperate through reverse engineering, using **your own** credentials
> and devices. Use at your own risk.

## Add the repository

1. **Settings → Add-ons → Add-on Store → ⋮ (top right) → Repositories**
2. Paste this URL and add it:
   ```
   https://github.com/Playcolors-co/ha-addons
   ```
3. The add-ons below appear in the store (listed under **Playcolors.co**).

## Add-ons in this repository

| add-on | description | docs |
